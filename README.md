# Last Light Mart / 最后一盏灯

移动端 H5 末世生存推理游戏。玩家在暴雨封锁的城市中调查人物、操作证物、组合线索并决定幸存者的命运。

## 运行

```powershell
python -m http.server 8080
```

访问 `http://localhost:8080/story.html`。

## 当前内容

- 第一、第二章完整剧情与多路线选择
- 自动对话、场景镜头和途中事件
- 可操作证物、证据组合与推断系统
- 资源、信任、人物关系和多结局
- 移动端竖屏界面、雨夜动态效果和本地存档

## 主要入口

- `story.html`：当前剧情游戏
- `fp.html`：第一人称场景原型
- `shop.html`：早期便利店经营原型
